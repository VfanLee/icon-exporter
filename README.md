# IconForge

把 SVG 快速合成 App 风格图标，并批量导出多种尺寸与格式。

适合制作应用图标、Dock 图标、多尺寸素材包等。

## 能做什么

- 上传或编辑 SVG
- 实时预览合成效果
- 调整容器背景、内边距、圆角
- 一次导出 PNG、WebP、JPEG、AVIF、SVG
- 同时输出多个导出尺寸，打包为 ZIP

## 怎么用

### 启动

需要 [Node.js](https://nodejs.org/) 环境：

```bash
pnpm install
pnpm dev
```

浏览器打开 http://localhost:5173

### 基本流程

1. **上传 SVG** — 点击「源码」里的「上传 SVG」，或在编辑器里粘贴代码
2. **预览调整** — 在「预览」里查看效果，可切换预览分辨率；右侧「容器」「图形」调整合成效果（预览与导出一致）；「导出」配置文件名、格式与尺寸
3. **导出** — 在右侧「导出」Tab 底部点击「导出 ZIP」下载成品
